# Public SDK

## Purpose

The VRP SDK provides public integration interfaces for future Pilot participants.

It is designed to simplify integration while preserving Protected Runtime isolation.

---

## Planned Components

Future SDK releases may contain:

- client interfaces
- request builders
- evidence helpers
- validation helpers
- compatibility utilities
- example integrations

---

## Design Principles

The SDK should remain:

- lightweight
- deterministic
- versioned
- implementation independent

---

## Important

The SDK is not the runtime.

It does not contain:

- runtime implementation
- protected algorithms
- authority logic
- recovery engine
- migration engine
