# Python SDK

## Overview

The Python SDK provides a lightweight interface for interacting with the VRP Public Engineering Boundary.

The SDK is intended for:

- engineers
- validation partners
- automation
- integration testing
- Pilot preparation

---

## Current Features

- Public Health
- Capability Discovery
- Validation Status
- Public Verdict
- Evidence Reference

---

## Not Included

The SDK intentionally excludes:

- Protected Runtime
- Runtime Algorithms
- Authority Engine
- Recovery Engine
- Runtime Mathematics
- Commercial Runtime

---
