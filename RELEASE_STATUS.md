# Release Status

## Current Release

Engineering Boundary v0.1

This repository represents the first public engineering boundary of the VRP ecosystem.

It intentionally excludes the Protected Runtime and all proprietary implementation details.

Future releases may expand the public engineering surface while preserving the separation between public interfaces and protected runtime components.

## Scope

Included:

- Documentation
- Public Contracts
- Public Schemas
- SDK Surface
- Integration Guidance
- Validation Concepts
- Evidence Concepts

Excluded:

- Protected Runtime
- Internal Algorithms
- Mathematical Models
- Runtime Decision Logic
- Commercial Runtime Components

Status: PUBLIC PREVIEW
