# Go Examples

## Purpose

This directory contains minimal examples demonstrating how to interact with the VRP Public Engineering Boundary using the official Go SDK.

Examples are intentionally limited to public engineering interfaces.

They are suitable for:

- engineering evaluation;
- integration testing;
- SDK familiarization;
- Pilot preparation.

No Protected Runtime functionality is included.

Current planned examples:

- health
- capabilities
- validation-status
- validation-verdict
- evidence-reference

Additional examples will be added in future releases.
