# Evidence

## Purpose

Evidence allows external verification of runtime behavior.

Evidence is an observable artifact.

It is not the runtime itself.

---

## Planned Evidence Types

- execution evidence
- validation evidence
- integrity evidence
- compatibility evidence
- pilot evidence

---

## Public Rule

Evidence may prove that an event occurred.

Evidence does not reveal how the runtime reached its decision.

---

Protected Runtime implementation is never included.
