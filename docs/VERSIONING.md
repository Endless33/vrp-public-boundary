# Versioning Policy

## Purpose

This repository follows explicit engineering versioning.

Every public document, contract, schema and SDK interface may evolve independently from the Protected Runtime.

---

## Public Versioning

Examples

Engineering Boundary v0.1

Engineering Boundary v0.2

SDK v1

Evidence Schema v1

Contract v1

---

## Compatibility

Breaking public changes should increment the major version.

Minor improvements should remain backward compatible whenever possible.

---

## Runtime

Protected Runtime versions are intentionally independent from this repository.
