# Public Disclosure Policy

## Principle

VRP follows a layered disclosure model.

Not every engineering artifact is public.

---

## Public

- documentation
- validation concepts
- engineering contracts
- schemas
- integration guidance
- evidence formats
- SDK interfaces

---

## Protected

- runtime implementation
- authority logic
- recovery algorithms
- migration engine
- mathematical models
- internal state machines
- commercial runtime

---

## Purpose

The objective is to enable technical evaluation without disclosing protected implementation.
