## Publication Rules

Only engineering artifacts suitable for public disclosure are published.

Protected implementation remains outside the lifecycle of this repository.

---

## Continuous Improvement

Each release may improve:

- documentation
- SDK interfaces
- contracts
- schemas
- validation
- deployment guidance
- compatibility

---

## Protected Runtime

The runtime follows its own internal engineering lifecycle.

It is intentionally separated from this repository.
