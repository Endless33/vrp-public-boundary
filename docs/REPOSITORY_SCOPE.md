# Repository Scope

This repository represents only the public engineering boundary of VRP.

---

## Included

- documentation
- engineering contracts
- public schemas
- SDK interfaces
- integration guides
- evidence specifications
- validation concepts

---

## Excluded

- Protected Runtime
- proprietary implementation
- internal mathematics
- authority engine
- recovery engine
- migration algorithms
- replay protection implementation
- private telemetry
- customer-specific runtime

---

## Objective

Allow engineers to understand:

- how to integrate
- how to validate
- how to verify

without exposing

- how the runtime actually works internally.
