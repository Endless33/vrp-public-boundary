# Public Schemas

This directory is reserved for future public schemas.

Possible schema categories:

- evidence
- verdicts
- capabilities
- compatibility
- pilot metadata
- public manifests

Schemas describe exported data only.

They never describe the internal implementation that generated that data.

---

## Allowed

- public fields
- public identifiers
- version numbers
- evidence references
- compatibility information

---

## Never Included

- runtime state
- internal algorithms
- authority transitions
- mathematical implementation
- protected structures
- security internals
- customer-specific logic

---
