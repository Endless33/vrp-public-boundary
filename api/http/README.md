# HTTP Examples

## Purpose

This directory contains raw HTTP request examples for the VRP Public Engineering Boundary.

The examples are suitable for:

- curl
- REST Client
- HTTPie
- Postman verification
- manual API exploration

---

## Included

- health.http
- capabilities.http
- validation.http
- verdict.http
- evidence.http

---

## Production Notice

Example endpoints are placeholders.

Authorized Pilot environments receive dedicated endpoint information separately.

---

## Boundary

Only public engineering interfaces are represented.

Protected Runtime functionality is intentionally excluded.
