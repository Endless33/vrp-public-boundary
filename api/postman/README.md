# Postman Collection

## Overview

This directory contains the official Postman collection for the VRP Public Engineering Boundary.

The collection allows engineers to explore the public engineering interfaces without requiring access to the Protected Runtime.

## Included Endpoints

- Health
- Capabilities
- Validation Status
- Validation Verdict
- Evidence Reference

## Production Notice

Example URLs included in this collection are placeholders.

Production endpoints are issued only to authorized Pilot environments.

## Purpose

The collection exists to simplify:

- API exploration
- SDK development
- Integration testing
- Public engineering validation

Protected Runtime functionality is intentionally excluded.
